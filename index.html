Desktop environment based on Fvwm

  • Klike Layout
    with default theme PitchBlack, a dark violet and plain theme and compositing.
  • Horizontal Layout
    with Tribal theme, held in green/black tones with yellow font collors and tribal signs as buttons.
  • Vertical layout
    with CrystalBall theme, a milky Mac OSX theme with white, grey and blue tones.
  • G2like layout
    with Clean, a simple vector theme with blue/gray color gradients for title bar and Fvwm modules.
Welcome to Fvwm-Nightshade! This project aims to be a lightweight but feature rich and good looking configuration of Fvwm. It contains many useful things to make work and life easier with a flavour of a modern desktop. But it can be configure by your own. It is good documented and shows you what is possible with this powerful window manager.

Latest News

Monday, 16th May 2016
After two long years I've finished FNS for the upcoming version 0.8. Now Beta stage is started.
Many new features have found the way into Fvwm-Nightshade:

  • Different layouts for any desktop resolution (up to 4K)
  • Many eye candy themes/decors changable on the fly without restart.
  • Localization support for German, French, Spanish and Russian.
  • Graphical config tools for different settings.
    • And much more
    Also the new website is available with many many new and updated information.

    Currently Fvwm-Nightshade Beta is downloadable over Master branch. Check it out if you like what you see!
